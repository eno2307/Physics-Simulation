ばねの運動をシミュレートするプログラムです。
「euler.c」がオイラー法、「mod_euler.c」が修正オイラー法、「runge_kutta.c」がルンゲクッタ法です。
「runge_kutta_2.c」は、粘性抵抗を加味したシミュレーションです。
「pendulum.c」は、ばねではなく振り子の運動をシミュレートしたものです。ルンゲクッタ法を用いています。

実行すると、時刻・質点の位置・速度をシミュレートした結果をcsvファイルに出力します。
その後はexcelなどお好みのツールでグラフ出力をしてください。

コード上部の「#define」部分を編集する事で、演算に関わるパラメータを変更できます。

出力されるファイルの場所を変更したい場合は以下を参照してください。
「euler.c」の場合：27行目の「"euler.csv"」をお好みのファイルパスに変更してください。
「mod_euler.c」の場合：29行目の「"mod_euler.csv"」をお好みのファイルパスに変更してください。
「runge_kutta.c」の場合：33行目の「"runge_kutta.csv"」をお好みのファイルパスに変更してください。
「runge_kutta_2.c」の場合：34行目の「"runge_kutta_2.csv"」をお好みのファイルパスに変更してください。
「pendulum.c」の場合：36行目の「"pendulum.csv"」をお好みのファイルパスに変更してください。
